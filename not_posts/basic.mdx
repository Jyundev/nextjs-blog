간단한 프로젝트를 구상해보기로 했다.
처음에는, 최근 저렴한 쇼핑몰에 대한 관심이 높아진 흐름에 맞춰, 저렴한 유사 상품을 추천해주는 앱을 만들어보려 했지만 데이터 구축에 어려움을 느껴, 회의 끝에 <b>간단한 메모/일기 앱</b>을 개발하기로 방향을 바꿨다.

우선은 빠르게 결과물을 내는 것을 목표로, Next.js 기반의 PWA 웹앱으로 제작해 Vercel에 배포하고, 이후 구글 플레이 스토어에도 배포하기로 했다.
기능은 많지 않지만, 추후 확장을 염두에 두고 빠르게 MVP를 만들어보는 것을 1차 목표로 정했다.

## 기능 정리

초기 MVP에 포함할 수 있는 기능들은 다음과 같다.

- 회원가입
- 로그인
- 일기 작성 / 수정 / 삭제
- 간단한 감정 태그
- 간단한 DB 연동

## 데이터베이스

PostgreSQL과 Supabase를 사용해 데이터베이스를 구성
Supabase는 Firebase처럼 간편하게 사용할 수 있는 오픈소스 백엔드 플랫폼으로,
PostgreSQL 기반의 데이터 저장과 인증, 실시간 기능까지 함께 제공해 개발 초기 세팅에 매우 유용했어요.
