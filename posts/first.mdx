## 블로그 기능 소개

이 블로그는 MDX를 활용해 다양한 콘텐츠를 쉽고 깔끔하게 표현합니다.

커스텀 링크로 내부와 외부 링크를 자동 구분하여 네비게이션을 간편하게 관리합니다.

코드 하이라이팅 기능으로 개발 관련 글에 가독성 높은 코드 블록을 제공합니다.

트위터 임베드를 통해 실시간 소식이나 인용을 자연스럽게 삽입할 수 있습니다.

테이블 컴포넌트로 정보를 명확하고 깔끔하게 정리해 보여줍니다.

---

## 🧭 커스텀 링크

내부 페이지 링크와 외부 링크를 각각 다음과 같이 처리할 수 있습니다:

- 내부 링크 예시: [블로그 홈으로 이동](/blog)
- 외부 링크 예시: [Next.js 공식 문서](https://nextjs.org/docs)

`CustomLink` 컴포넌트는 자동으로 `/`로 시작하는 링크는 `<Link>` 처리, 그 외는 외부 링크로 처리합니다.

---

## ✨ 코드 하이라이팅

MDX에서 코드 블록 작성 시 자동으로 구문 하이라이트가 적용되어 가독성을 높입니다.
예를 들어, 슬러그 생성 함수는 다음과 같습니다:

```js
function slugify(str) {
  return str
    .toLowerCase()
    .trim()
    .replace(/\s+/g, "-")
    .replace(/[^\w\-]+/g, "")
    .replace(/\-\-+/g, "-");
}
```
