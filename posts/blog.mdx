# Vercel의 Portfolio Starter Kit

Portfolio Starter Kit은 Next.js 기반의 개인 포트폴리오와 블로그를 빠르게 만들 수 있는 공식 템플릿이다.
이 템플릿을 기반으로 블로그를 새로 구축할 예정이다.

기존에 [Jekyll + GitHub Actions 블로그](https://jyundev.github.io/)가 있었지만, Next.js 공부도 할 겸 직접 만들어보고 싶었다.

## MDX 란?

MDX는 Markdown과 JSX를 합친 문서 포맷이다. Markdown 문법 안에 React 컴포넌트(JSX)를 바로 넣을 수 있게 해준다.

MDX를 사용하면 컴포넌트 기반으로 블로그를 작성할 수 있다는 점이 가장 큰 장점이다.

현재 이 테이블도 컴포넌트를 활용해 구현한 예시이다.

## SEO 최적화

Next.js의 서버사이드 렌더링과 동적 메타 태그 설정 기능을 활용해 SEO(검색엔진 최적화)가 잘 되어 있다. 페이지마다 동적으로 제목, 설명, Open Graph 이미지 등을 설정할 수 있어서 검색 결과에서 노출이 잘 되고, SNS 공유 시에도 예쁜 미리보기 이미지를 보여준다.

---

Tailwind CSS도 제대로 써본 적 없었지만, 블로그를 만들면서 제대로 공부해 볼려고 한다. 내가 원하는 스타일과 기능을 가진 기술 블로그로 점차 바꿔나갈 계획이다.
