# MDX로 리치한 블로그 콘텐츠 만들기

Next.js에서 **MDX**를 활용하면 마크다운 기반의 글에 React 컴포넌트를 쉽게 포함시킬 수 있습니다.  
이 글에서는 커스텀 컴포넌트를 이용해 블로그 콘텐츠를 더욱 풍성하게 만드는 방법을 소개합니다.

---

## 🧭 커스텀 링크

내부 페이지 링크와 외부 링크를 각각 다음과 같이 처리할 수 있습니다:

- 내부 링크 예시: [블로그 홈으로 이동](/blog)
- 외부 링크 예시: [Next.js 공식 문서](https://nextjs.org/docs)

`CustomLink` 컴포넌트는 자동으로 `/`로 시작하는 링크는 `<Link>` 처리, 그 외는 외부 링크로 처리합니다.

---

## ✨ 코드 하이라이팅

MDX 내에서 코드 블럭을 입력하면 `highlight`를 통해 구문 강조가 적용됩니다.  
예를 들어, 슬러그를 만드는 자바스크립트 함수는 다음과 같습니다:

```js
function slugify(str) {
  return str
    .toLowerCase()
    .trim()
    .replace(/\s+/g, "-")
    .replace(/[^\w\-]+/g, "")
    .replace(/\-\-+/g, "-");
}
```
